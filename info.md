# Vertical Mixer Card

Mixer-console style vertical sliders for any `number` or `input_number`
entity in Home Assistant.

- Group entities side by side like a real mixing desk
- Reads `min` / `max` / `step` / `unit` from each entity automatically
- Tick marks on both sides of the track, mirroring a real fader
- Optional unity / "0 dB" reference line
- Per-entity accent color (mushroom palette names or raw CSS colors)
- Touch friendly, Shadow DOM scoped, theme aware
- Visual editor included
